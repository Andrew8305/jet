Jet

Lightweight, Realtime Message Bus for the Web

Don't waste time

Build Realtime Apps now! Jet is a framework for building distributed realtime Apps. It employs fast push technologies and differential updates to keep all clients in sync. It is very easy to use and features powerful query filters, sorting rules and a Web Inspector.

Jet is suitable for IoT , Games, Collaborative Apps or anything else where things are in flux and fast communication is required. Its efficient design allows Jet Servers to run very nicely on systems with rather limited ressources like the RaspberryPi.

Make your choice

Jet is free for all! And of course, everything is open source. Beside the HTML5 based free client there are free servers available for node.js and Lua.

Jet is meant for Self-hosting. Intergration is very easy as Jet servers can run side-by-side with your webserver environment or fully integrate with your node.js based solutions if desired.

Take that bus

Besides its web-capable realtime characteristics, Jet is intended to be a system bus for Linux, Windows or OSX. Much like the widely used Dbus, Jet allows distribution of system services to muliple processes using the same simple Jet API.

If you take the Jet Bus, your services are available for Browsers instantly!

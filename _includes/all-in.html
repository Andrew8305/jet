Build Realtime Apps now! Jet is a framework for building distributed realtime Apps. It employs fast push technologies and differential updates to keep all clients in sync. It is very easy to use and features powerful query filters, sorting rules and a Web Inspector.

Jet is suitable for IoT , Games, Collaborative Apps or anything else where things are in flux and fast communication is required. Its efficient design allows Jet Servers to run very nicely on systems with rather limited ressources like the RaspberryPi.
